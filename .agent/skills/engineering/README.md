# Engineering Skills

Skills for software engineering workflows.

| Skill | Trigger mode | Description |
|-------|-------------|-------------|
| `diagnose` | auto | Disciplined diagnosis loop for hard bugs and performance regressions |
| `tdd` | auto | Test-driven development with red-green-refactor loop |
| `to-issues` | auto | Break a plan or PRD into independently-grabbable issues |
| `to-prd` | auto | Turn current conversation context into a PRD |
| `triage` | auto | Triage issues through a state machine driven by triage roles |
| `grill-with-docs` | user-only | Grilling session against the existing domain model and documentation |
| `zoom-out` | user-only | Zoom out and map the relevant modules and callers |
| `improve-codebase-architecture` | auto | Find deepening opportunities in a codebase |
| `setup-maf-skills` | user-only | Configure issue tracker, triage labels, and domain docs for this repo |

Run `/setup-maf-skills` before first use of `to-issues`, `to-prd`, `triage`, `diagnose`, `tdd`, `improve-codebase-architecture`, or `zoom-out`.
